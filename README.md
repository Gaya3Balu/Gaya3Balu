### 🏥 Vendor Compliance Dashboard & SLA Monitoring
`Power BI` `Python` `SQL` `Excel`

Built a centralized compliance dashboard replacing fragmented Excel trackers across vendor teams.

| Metric | Result |
|---|---|
| SLA breach identification time | ↓ **35%** |
| Manual reporting effort | ↓ **40%** |
